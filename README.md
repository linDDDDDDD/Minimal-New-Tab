# Minimal New Tab - 极简新标签页

一个美观、功能丰富的Edge浏览器新标签页扩展

## ✨ 功能特性

### 🔍 搜索功能
- 支持百度、Google、Bing三种搜索引擎
- 智能搜索建议
- 历史记录联想
- 图片搜索功能

### 🎨 壁纸与主题
- 自定义壁纸上传(最多4张)
- 壁纸自动轮换
- 跟随壁纸自动提取主题色
- 浅色/深色主题切换

### 🔗 快捷导航
- 可添加常用网站快捷方式
- 自动获取网站图标
- 可收起侧边栏

### 📅 日历插件
- 显示当前月份日历
- 可开关显示

### 📚 课表插件
- PDF课表解析
- 手动添加/编辑课程
- 支持学校名称设置
- 周数自动计算

## 🚀 安装使用

### 开发者模式安装
1. 下载或克隆此项目
2. 打开Edge浏览器,访问 `edge://extensions/`
3. 开启「开发人员模式」
4. 点击「加载解压缩的扩展」
5. 选择项目文件夹

### 常规使用
- 安装后打开新标签页即可使用
- 点击右下角设置按钮可进行个性化配置

## 📁 项目结构

```
edge插件/
├── manifest.json      # 扩展配置文件
├── index.html         # 主页面
├── style.css          # 样式文件
├── script.js          # 核心逻辑
└── 课表模板.html      # 课表模板
```

## 💡 技术亮点

- **Manifest V3**: 采用最新的扩展规范
- **IndexedDB**: 高效存储壁纸数据
- **Canvas API**: 提取壁纸主色调
- **LocalStorage**: 存储用户配置
- **原生JavaScript**: 无依赖,轻量高效

## 🔧 配置说明

### 搜索引擎
在设置中可选择默认搜索引擎:
- Bing (默认)
- Google
- 百度

### 壁纸设置
- 支持上传JPG/PNG等常见图片格式
- 开启轮换后,每次打开新标签页自动切换
- 最多可保存4张壁纸

### 课表使用
1. 使用项目中的「课表模板.html」填写课程信息
2. 保存为PDF文件
3. 在设置-课表中上传PDF
4. 或直接手动添加课程
